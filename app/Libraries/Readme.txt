How to use:

	1- Run Onyx WhatsApp Client application (If your antivirus blocks the program, Allow it to run or disable your antivirus)
	2- Find your User ID and API Key from Dashboard (on www.onyxberry.com/services/wapi) and enter your credentials when asked in Onyx WhatsApp Client.
	3- After running new file named "chromedriver.exe" will be created, its important for the Onyx WhatsApp Client to
	  communicate with your Google Chorme.
	  
	4- If your antivirus blocks the program to run, disable your antivirus or allow the program to run.
